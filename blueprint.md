# MonkeysParadise 002: 글로벌 트렌드 뷰어

## 개요

이 프로젝트는 구글 트렌드에서 가져온 것과 유사하게 다양한 카테고리의 주요 글로벌 트렌드를 표시하는 간단한 웹 애플리케이션입니다.

## 기능

*   **메인 페이지:** 깔끔하고 단순한 인터페이스.
*   **반응형 디자인:** 모바일 친화적인 레이아웃입니다.
*   **자동 생성:** 페이지 로드 시 자동으로 트렌드 시각화를 생성합니다.

## 이전 작업
- **v1 (리스트 뷰):** 분야별 트렌드 키워드를 리스트 형태로 구현.
- **v2 (그리드 뷰):** UI를 그리드 레이아웃으로 변경하고 대표 이미지 추가.
- **v3 (단순 그리드 뷰):** 카테고리 표시 제거 및 UI 간소화.
- **v4 (버블 차트):** D3.js를 이용해 100개 키워드를 인터랙티브한 버블 차트로 구현.
- **v5 (자동화):** 페이지 로드 시 차트 자동 생성 및 버블 크기 편차 조정.
- **v6 (다크 모드):** cryptobubble.com 스타일을 모방한 다크 모드 및 버블 디자인 적용.

## 현재 작업: D3.js 트리스마일(Treemap) UI로 변경

사용자가 버블 차트 대신, 검색량에 따라 크기가 다른 사각형이 간격 없이 배열되는 '바둑판' 형태의 UI를 요청했습니다. 이는 D3.js의 트리스마일(Treemap) 레이아웃에 해당합니다.

### 계획:

1.  **`blueprint.md` 업데이트:** 현재 요청사항을 반영하여 계획을 수정합니다. (완료)
2.  **`main.js` 수정:**
    *   `displayBubbleChart` 함수를 `displayTreemap`으로 이름을 변경하고 로직을 전면 수정합니다.
    *   **레이아웃 변경:** D3.js의 `d3.pack()` 대신 `d3.treemap()`을 사용합니다.
    *   **타일링 설정:** 보기 좋은 사각 형태를 위해 `d3.treemapSquarify` 옵션을 사용합니다.
    *   **간격 제거:** 사용자의 요청에 따라 `padding` 관련 설정을 모두 0으로 하여 사각형들이 서로 붙어있게 만듭니다.
    *   **렌더링 로직 변경:** `<circle>` 대신 `<rect>`를 렌더링하도록 코드를 변경합니다.
    *   **텍스트 처리:** 사각형의 크기가 작아 텍스트가 보이지 않을 경우를 대비해, 텍스트를 숨기거나 잘라내는(clipping) 로직을 추가하여 깔끔한 UI를 유지합니다.
    *   **상호작용:** 기존의 구글 트렌드 링크 기능은 그대로 유지하며, 호버 효과는 사각형에 맞게 단순화합니다.
3.  **`style.css` 수정:**
    *   기존 버블 차트 관련 스타일(필터 등)을 제거하고, 트리스마일 텍스트 표시에 필요한 최소한의 스타일을 조정합니다.
